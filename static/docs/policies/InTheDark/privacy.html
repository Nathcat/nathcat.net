In The Dark Privacy Policy

This document details the use and handling of data in "In The Dark", a game developed by Nathcat 2020.

The game does not access personal user data, the only data that is accessed and transported elsewhere are logs of crashes that occur, these are delivered to the developer through Google Play. The data included in these reports include the device type, and a stack trace of the error that caused the crash, but does not include any personal user data

The game itself passes on no other data to the developer.
